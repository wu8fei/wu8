# 我的世界暮色机械服推荐网页

## 项目简介

这是一个为"我的世界暮色机械服"设计的推荐网页，用于展示服务器相关信息，方便玩家了解并加入服务器。网页采用现代化设计，具有响应式布局，适配各种设备屏幕。

## 功能特点

- 🎮 展示服务器图标和基本信息
- 📱 显示QQ群二维码，方便扫码加入
- 📋 一键复制QQ群号和服务器地址
- 🎨 美观的界面设计，采用《我的世界》暮色森林主题背景
- 📱 响应式布局，适配电脑、平板、手机等设备

## 技术栈

- HTML5
- CSS3 (使用CSS变量实现主题管理)
- JavaScript (原生JS实现交互功能)
- Font Awesome (图标库)

## 页面结构

- **顶部标题区**：展示服务器名称和欢迎语
- **服务器图标展示区**：展示服务器图标和简短描述
- **QQ群信息展示区**：包含二维码、群号和复制功能
- **服务器地址展示区**：显示服务器地址和复制功能
- **底部说明区**：版权信息

## 使用方法

1. 打开 `index.html` 文件即可访问网页
2. 扫描QQ群二维码或复制群号加入服务器交流群
3. 复制服务器地址，在《我的世界》中添加服务器
